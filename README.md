# GameServer Manager (GSM)

A PowerShell framework for managing dedicated game servers on Windows.

Status: v0.1.0-alpha, Phase 1 in progress. Phase 1 supports Insurgency (2014),
Team Fortress 2, Counter-Strike: Source, Left 4 Dead, and Left 4 Dead 2.

## Requirements

- Windows 11
- PowerShell 7+
- SteamCMD (installed automatically by `Core/SteamCMD.psm1`)

## Quick start

```powershell
./GSM.ps1
```
